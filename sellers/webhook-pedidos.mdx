---
title: "Webhook de pedidos"
description: "Como receber, via webhook, o payload enviado pela Mkplace ao seller a cada evento de pedido."
---

Para que o vendedor receba as atualizações de um pedido, é necessário criar um endpoint que receba um payload específico — e, a partir dele, armazenar os dados relevantes e tomar as ações necessárias de forma tempestiva.

O endpoint deve aceitar:

| Atributo | Valor |
|---|---|
| **Método** | `POST` |
| **Header** | `Content-Type: application/json` |

<Card title="Baixar exemplo de payload" icon="download" href="https://raw.githubusercontent.com/mkplace/docs/main/examples/exemplo-pedido-sales.zip">

</Card>
